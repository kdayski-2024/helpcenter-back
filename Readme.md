## helpcenter-back

Backend-сервис справочного/ help-center приложения на Node.js и Express с использованием PostgreSQL и ORM Sequelize.

### Основные возможности

- **REST API**: эндпоинты для работы со статьями, категориями и прочим контентом справочного центра.
- **База данных**: PostgreSQL, управление схемой через миграции (`sequelize-cli` и скрипты в `database/migrations`).
- **Загрузка файлов**: поддержка загрузки файлов через `multer`.
- **Email-уведомления**: отправка писем через `nodemailer`.
- **Оплата**: интеграция со Stripe (если используется в конкретной установке проекта).

### Требования

- **Node.js**: рекомендовано не ниже 18.x
- **npm** или **yarn**
- **PostgreSQL**: установленный и доступный экземпляр БД

### Установка

```bash
npm install
```

Создайте файл окружения `.env` в корне проекта на основе ваших настроек (пример переменных):

```bash
PORT=30002
DATABASE_URL=postgres://user:password@localhost:5432/helpcenter
NODE_ENV=development
```

Формат и точный набор переменных может отличаться в зависимости от вашей конфигурации.

### Миграции и сиды БД

- **Создать миграцию**:

```bash
npm run db:make
```

- **Применить миграции**:

```bash
npm run db:migrate
```

- **Отметить состояние миграций**:

```bash
npm run db:setmigrate
```

- **Заполнить БД начальными данными**:

```bash
npm run db:seed
```

### Запуск проекта

- **Режим разработки** (с `nodemon` и debug-логами):

```bash
npm run dev
```

- **Обычный запуск**:

```bash
npm start
```

- **Отладочный режим с расширенным логированием**:

```bash
npm run debug
```

### Прочие скрипты

- **Освободить порт 30002** (Linux/macOS, при наличии утилиты `kill-port`):

```bash
npm run killport
```

При необходимости скорректируйте порт в `.env` и в скрипте.

### Структура проекта (общее представление)

- `index.js` — точка входа HTTP-сервера.
- `database/` — миграции, сидер и вспомогательные скрипты для работы с БД.
- `models/`, `controllers/`, `routes/` (если присутствуют) — слои данных, бизнес-логики и маршрутизации.

Фактическая структура может немного отличаться, ориентируйтесь по содержимому директории.
